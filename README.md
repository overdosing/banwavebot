# discord ban wavebot
enables the author to prepare a list of users for a scheduled ban wave instead of applying bans immediately.

## installation
prerequisites - 

- [python 3.8+](https://www.python.org/)
- [nextcord 3.0+](https://github.com/nextcord/nextcord)

after installing the prerequisites, execute `banwavebot.py`
